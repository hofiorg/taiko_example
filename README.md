# First Tests with Taiko ![node_js workflow](https://github.com/hofiorg/taiko_example/actions/workflows/node.js.yml/badge.svg)

## Installation

```sh
git clone https://github.com/hofiorg/taiko_example.git
cd taiko_example
npm install
```

## Start

```sh
npm start
```

## Test

```sh
npm test
```
