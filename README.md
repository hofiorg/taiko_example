Example project demonstrating how to write browser automation tests with [Taiko](https://taiko.dev).

## 📥 Installation

```sh
git clone https://github.com/hofiorg/taiko_example.git
cd taiko_example
npm install
```

## 🚀 Start

```sh
npm start
```

## 🧪 Test

```sh
npm test
```
